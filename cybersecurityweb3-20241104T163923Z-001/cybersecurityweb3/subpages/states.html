there are 3 different states of data. you need to know the state of the data to apply the correct form of cybersecurity:
  • data at rest - stored information, usually on databases, hard drives, flash drives, memory, dvds, etc.
  • data in transit - information being transferred, being moved from one system to another such as file sharing on LAN, transfer on internet, etc.
  • data in use - data being processed, such as data creation, data being used in an app, and data being placed in memory
