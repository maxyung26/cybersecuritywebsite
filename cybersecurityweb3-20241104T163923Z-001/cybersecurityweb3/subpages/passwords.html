passwords are an authentification method that can be exploited if not used properly
short passwords with limited number of characters (ex: only numbers) will be able to be guessed by brute force by hackers quicker than more complex passwords with different letters and characters

there are 3 types of password attacks
  • stealing - hacker steals the password by seeing what the user types in either by looking over their shoulder, keylogging, etc.
  • brute force - hacker uses a program to guess every combination of characters to get a valid password
  • stupid user - hacker uses information they know about the user to guess the password (birthdays, pet names, etc.)
